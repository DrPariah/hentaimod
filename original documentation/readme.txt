====================
はじめに
====================
公式のCDDA本体でも動作します。そのはずです。

0.C-8204でテストプレイしています。それ以降でもたぶん動くと思います。（無責任）
NPCの会話周りとか重量の記述とか、わりと最近に実装された要素を使ってるので、それなりに古いバージョンだとエラーが発生するかもしれません。ご注意を。


====================
導入方法
====================
「Hentai」フォルダをcataclysmdda\data\modsに配置して下さい。
※フォルダ名を変更すると正常に動作しません。もしも変えたい場合はpreload.luaの頭らへんにあるフォルダパスをいい感じに編集して下さい。UTF-8形式でね！


====================
このMODについて
====================
・NPCやペットとの*気持ちいいこと*ができるようになります。
・*気持ちいいこと*を仕掛けてくる敵やダンジョンが追加されます。
・*気持ちいいこと*により妊娠するようになります。避妊していなければの話ですが。
・HENTAI的なアイテムや特質がいろいろと追加されます。
・特定のペットをNPCに変化させることができるようになります。擬人化的な感じ。
　それ専用の職業（「一人と一匹」）を一つ追加したので手っ取り早く試してみたい方はどうぞ。


NPCと*気持ちいいこと*を行う場合、アイテム「避妊具」をNPCに対して使用して下さい。
対象に対して強い影響力を持っていれば応じてくれるはずです。


18禁要素はありません。でも15禁くらいかもしれません。法律とご自身のリスクを考えてご活用下さい。
あと、手とキーボードは清潔にしましょう。
